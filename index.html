The aim of the game is to catch the carrot 20 times before the timer gets to 0

Be careful not to run into the foxes as you will loose a life

If you lose all 5 lives it GameOver

Take care and dont bump into any obstacles or you will go back to the start

if you are on a laptop use the arrow keys to move the rabbit

if using a mobile device use buttons below game screen

if using a tablet or mobile device please play in portrait mode!!!
